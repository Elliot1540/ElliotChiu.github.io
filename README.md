# Jichun Zhao (Elliot Chiu) Academic Website

This repository contains the AcademicPages-based personal academic website for Jichun Zhao (Elliot Chiu), a PhD Candidate at HIT Lab NZ, University of Canterbury.

## Local Preview

Install the Ruby dependencies and run the Jekyll server:

```bash
bundle install
bundle exec jekyll serve
```

The site is configured for GitHub Pages at:

```text
https://elliot1540.github.io/ElliotChiu.github.io/
```

## Content

- Homepage: `_pages/about.md`
- Publications: `_publications/`
- Projects: `_portfolio/`
- CV: `_pages/cv.md`
- Site configuration: `_config.yml`
- Navigation: `_data/navigation.yml`
